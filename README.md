# Music Player Frontend

The frontend for [Music Player Backend](https://github.com/jaydp17/music-player-backend), built using React/Redux.

## ✨ Features

* Streams songs instead of download
* Uses WebSockets to live track the number of people listening to a song

## 💾 Pre-requisites

* [node.js](https://nodejs.org/) v8.9+

## ▶️ Usage

```sh
# install npm deps
$ yarn

# run it locally
# yarn start

# build for production
# yarn build
```
